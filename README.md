# Recompact

[![build status](https://img.shields.io/travis/neoziro/recompact/master.svg?style=flat-square)](https://travis-ci.org/neoziro/recompact)

A set of React high order components based on Rx. It's a tiny framework inspired by recompose ton unleash the power of Rx inside your React application.

```sh
yarn add recompact
```
